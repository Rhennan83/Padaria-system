# 🏪 Padaria System

Sistema de gerenciamento de padaria com arquitetura OO refundida, blindada contra vulnerabilidades e 100% testado.

**Status**: ✅ Pronto para produção  
**Versão**: 1.1.0  
**Linguagem**: TypeScript  
**Node.js**: >= 14

---

## 🎯 Sobre o Projeto

Sistema de PDV (Ponto de Venda) para padarias com:
- ✅ Cadastro de produtos
- ✅ Registro de vendas
- ✅ Balancete de vendas
- ✅ Validação robusta em todas as entradas
- ✅ Testes automatizados integrados
- ✅ Persistência em arquivo JSON
